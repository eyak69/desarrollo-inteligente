# Prompt para migrar pantalla existente

Migrar la pantalla [NOMBRE] al design system oficial.

Reglas:
- No cambiar lógica de negocio.
- No cambiar endpoints.
- No cambiar permisos.
- Solo migrar presentación visual.
- Usar componentes oficiales.
- Mantener comportamiento actual.
- Hacerla mobile-first.

Antes:
- Listar archivos afectados.
- Explicar riesgos.

Después:
- Cómo probar.
- Qué quedó pendiente.
