# Mobile First

Toda pantalla debe funcionar primero en celular.

## Reglas obligatorias

- Diseñar desde 360px de ancho.
- No depender de hover.
- Botones táctiles cómodos.
- Inputs grandes y claros.
- Acciones importantes visibles.
- Menús colapsables.
- Tablas adaptadas a mobile.
- Formularios en una columna en mobile.
- Evitar modales gigantes en celular.
- Usar drawer o pantalla completa para edición compleja.

## Breakpoints sugeridos

- Mobile: 360px - 767px
- Tablet: 768px - 1023px
- Desktop: 1024px+

## Grillas en mobile

Para grillas complejas:
- Mostrar cards por registro.
- Mostrar campos clave.
- Usar botón "Ver detalle".
- Evitar 15 columnas apretadas.
