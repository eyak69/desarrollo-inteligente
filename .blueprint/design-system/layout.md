# Layout oficial

## Desktop

- Sidebar fijo a la izquierda.
- Header superior dentro del área principal.
- Área de contenido con padding consistente.
- Formularios en paneles o cards.
- Tablas ocupan ancho disponible.
- Acciones principales arriba a la derecha.

## Mobile

- Sidebar se convierte en drawer/hamburger.
- Header compacto.
- Acciones principales visibles.
- Tablas grandes deben tener versión mobile:
  - cards,
  - columnas críticas,
  - detalle expandible,
  - filtros simples.

## Regla

Todas las pantallas usan AppShell.
No crear layouts alternativos sin autorización.
